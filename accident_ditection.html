Road Accident Detection and Alert System

Abstract

Road accidents are a major concern due to careless driving, poor safety awareness, and increasing traffic. This project aims to provide an accident detection system using impact force sensing and alert generation.

Problem Statement

Rising accidents are caused by rash driving, poor roads, and violation of rules. This leads to injuries, deaths, and property loss affecting families and society.

Implications

  • Threat to life: More injuries and deaths.
  • Financial loss: Vehicle and hospital costs rise.
  • Traffic chaos: Roads get blocked.
  • Hospital load: More emergency cases.
  • Government burden: Increased safety spending.

System Working

The system detects accidents using an impact sensor. If threshold is crossed:

  1. Accident is detected
  2. GPS location is recorded
  3. Alert message is automatically sent

C Code Implementation

#include <stdio.h>
#include <string.h>

struct AccidentReport {
    float impactForce;
    char location[50];
    char alertMessage[100];
};

int detectAccident(float impact) {
    return (impact > 50.0);
}

void getGPSLocation(char loc[]) {
    strcpy(loc, "Latitude: 18.5204, Longitude: 73.8567");
}

void sendAlert(struct AccidentReport report) {
    printf("\n=== EMERGENCY ALERT SENT ===\n");
    printf("Impact Force: %.2f\n", report.impactForce);
    printf("Location: %s\n", report.location);
    printf("Message: %s\n", report.alertMessage);
}

int main() {
    struct AccidentReport report;
    float force;

    printf("Enter Impact Force: ");
    scanf("%f", &force);

    report.impactForce = force;

    if (detectAccident(force)) {
        getGPSLocation(report.location);
        strcpy(report.alertMessage,
               "Accident detected! Send ambulance!");

        sendAlert(report);
    } else {
        printf("No accident detected.\n");
    }
}

Conclusion

The accident detection and alert system reduces emergency response time and helps save lives. This project promotes safer transportation by quickly informing authorities in case of a crash.
